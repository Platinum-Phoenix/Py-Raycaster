# Installtion
Just download the source-code and run Main.py

# Dependencies

You need Python 3

[Download](https://www.python.org/downloads/release/python-385/)

You need PyOpenGL

install it via pip

``` bash
pip3 install PyOpenGL
```

or

```bash
pip install PyOpenGL
```
